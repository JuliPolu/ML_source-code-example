Setup Instructions
1. Clone the repository:
    ```bash
    git clone git@github.com:/JuliPolu/ml_example.git
    cd ml_example
    ```
2. Create a project:
    ```bash
    uv python pin 3.11.12
    uv init
    uv venv
    uv add --requirements requirements.txt
    ```
